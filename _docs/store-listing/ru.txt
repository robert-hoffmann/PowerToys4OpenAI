PowerToys for OpenAI™ предоставляет функциональность ChatGPT через официальный API OpenAI™

Предназначен для опытных пользователей

* Поисковый компаньон интегрирован с результатами поисковых систем
* Менеджер диалогов - это полноценный интерфейс чата с управлением историей
* Менеджер профилей позволяет точно настроить тип ответа модели
* Просмотр количества токенов
* Оценка цены разговора
* Копирование результатов в буфер обмена
* Поддержка рендеринга в формате Markdown
* Поддержка распознавания речи
* ... и многое другое в будущем ;-)

Оптимизирован для производительности и запускается локально в вашем браузере
(без затрат на сервер)

Для использования этого расширения необходимо иметь собственный API-ключ, который можно получить непосредственно у OpenAI, зарегистрировавшись и посетив их панель разработчика

Управление ключами API осуществляется на сайте:
https://platform.openai.com/account/api-keys

Цена составляет около $0.002 за 750 слов, или $1 за эквивалент содержания книги в 1000 страниц
(для модели gpt-3.5-turbo)
