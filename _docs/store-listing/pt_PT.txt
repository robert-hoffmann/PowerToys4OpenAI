PowerToys for OpenAI™ fornece funcionalidade ChatGPT através da API oficial da OpenAI™

É destinado para usuários avançados no escritório

* Search Companion está integrado aos resultados do motor de busca
* Conversation Manager é uma interface completa de chat com gerenciamento de histórico
* Profile Manager permite ajustar o tipo de resposta do modelo
* Veja a contagem de tokens
* Veja a precificação da conversa
* Copie os resultados para a área de transferência
* Suporte à renderização Markdown
* Suporte a reconhecimento de voz
* ... e mais por vir ;-)

É otimizado para velocidade, ao ser executado localmente em seu navegador
(sem custos de servidor)

O uso desta extensão requer ter sua própria chave de API, que pode ser obtida diretamente da OpenAI ao se registrar e visitar o painel de desenvolvedor

Gerencie suas chaves API visitando:
https://platform.openai.com/account/api-keys

O preço fica em cerca de $0,002 por 750 palavras, ou $1 pelo conteúdo equivalente a um livro de 1000 páginas
(para o modelo gpt-3.5-turbo)
